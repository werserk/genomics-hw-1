## Подготовка проекта к запуску


### 0) Установить `uv` (если отсутствует)

Linux/macOS:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### 1) Создать виртуальное окружение и установить зависимости

В корне репозитория:

```bash
uv sync --dev
```

### 2) Запуск

Можно запускать [HW1.ipynb](notebooks/HW1.ipynb)
